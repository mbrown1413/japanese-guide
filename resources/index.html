Listening

  • DownSub - Download subtitles from videos (youtube, etc.)
  • Language Reactor - Chrome extension for Netflix subtitles
    • Formally: Language Learning With Netflix
    • Some user reviews / comparisons w/ subadub here: https://www.reddit.com/r/ajatt/comments/eqyxkj/language_learning_with_netflix_an_alternative_to/ Chrome only extensions:
    • Language Learning with Netflix
    • Language Learning with Netflix & YouTube
  • LiveTL - Tool for following youtube livestream chats in other languages.
  • Nihongo Con Teppei
    • Beginner: http://nihongoconteppei.com/ (Nihongo con Teppei for beginners)
      • Tip: Start from episode one. It slowly gets harder as it goes on.
    • More advanced: http://teppeisensei.com/
    • Use DownSub to download subtitles.
